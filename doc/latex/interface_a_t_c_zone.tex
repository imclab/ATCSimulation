\subsection{\-Detailed \-Description}
\-A class representing a zone in the environment. \-A zone if composed of borders, which are segments instance of the class \-Zone\-Border\-Segment. \-Each zone has a controller, which is either an \hyperlink{interface_airport_controller}{\-Airport\-Controller} or a \hyperlink{class_zone_controller}{\-Zone\-Controller}.
